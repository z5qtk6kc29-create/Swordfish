This program is a game where you can control a swordfish. The objective is to poke the sharks without accidentally hurting the peaceful fish. You can play indefinitely until the shark bites you! If you poke a shark, you receive a point but if you hurt the fish, you will lose points

You will need to create a venv within the folder by running python -m venv venv
Then you will need to run pip install -r requirements.txt
